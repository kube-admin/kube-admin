# 贡献者公约

我们承诺为社区中每一位成员提供友好、安全、欢迎的体验，无论其经验、性别、性取向、身体状况、外貌、种族、民族、宗教信仰、国籍或技术水平。

## 我们的准则

营造正向环境的行为示例：

- 友善、包容，尊重不同观点与经验
- 优雅地接受建设性批评
- 关注社区整体利益

不可接受的行为：

- 骚扰、人身攻击、敌意或侮辱性言论
- 公开或私下骚扰他人
- 未经明确许可泄露他人私人信息
- 其他违反职业道德的不当行为

## 维护者的职责

项目维护者有权移除、编辑或拒绝不符合本公约的评论、提交、代码、Wiki 编辑、Issue 与其他贡献。维护者将本着善意、公正地执行本公约。

## 适用范围

本公约适用于所有项目空间（代码仓库、Issue、PR、文档站）以及公共空间中个人代表项目时的公开行为。

## 执行

如遇违反本公约的行为，请通过 GitHub Issue 或邮件联系维护者团队。所有举报都会被审慎评估并保密处理。

本公约改编自 [Contributor Covenant](https://www.contributor-covenant.org/) v2.1。
